NAME
    cls - clear the terminal screen

SYNOPSIS
    cls

DESCRIPTION
    The cls command clears the terminal screen and moves the cursor to the top-left corner.
    It is commonly used to refresh the display by removing previous output, providing a clean screen 
    for new output.
    This command is equivalent to the 'clear' command in Linux/Unix-like systems.

EXAMPLES
    - Clear the terminal screen:
        cls

SEE ALSO
    clear - clear terminal with Linux/Unix

NOTES
    The cls command usually only clears the visible screen.
